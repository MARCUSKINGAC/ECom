# ShopEasy - E-Commerce Website

A modern e-commerce website built with HTML, CSS, and JavaScript. ShopEasy allows users to browse products by category, view product details, and shop for various items across multiple categories.

## 📁 Project Structure

```
Qoder(Refine_New)/
├── index.html              # Home page with hero section and categories
├── products.html           # Products listing page with filtering
├── product-detail.html     # Individual product detail page
├── css/
│   └── style.css           # Main stylesheet
├── js/
│   ├── data.js             # Product dataset (36 products)
│   ├── main.js            # Home page functionality
│   ├── products.js        # Products page functionality
│   ├── product-detail.js  # Product detail page functionality
│   └── images/            # Product images folder
└── README.md              # This file
```

## 🛍️ Features

- **Home Page**: Hero section with welcome message and category navigation
- **Products Page**: Browse all products with category filtering
- **Product Detail**: View detailed information about each product
- **Category Filtering**: Filter products by Electronics, Clothing, Accessories, Sports & Games, Food & Beverages, Books
- **Responsive Design**: Works on desktop and mobile devices

## 🚀 Getting Started

1. **Open the project**: Navigate to the project folder in your browser
2. **Launch**: Open `index.html` in any web browser
3. **Browse**: Explore the home page, navigate to products, and view product details

## 🛠️ Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Styling and responsive design
- **JavaScript (ES6+)**: Dynamic functionality and data handling
